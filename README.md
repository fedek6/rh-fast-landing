# rh-fast-landing
Template for creating fast HTML5 lading pages (powered by Bootstrap 4, jQuery, wow.js)

```
git clone https://github.com/fedek6/rh-fast-landing
cd rh-fast-landing
npm install
grunt assets
grunt images
grunt packages
cd public
```


And it's there! Saaaay what? Your website is in public dir :)
